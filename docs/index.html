PaperTodo

让桌面上有几张安静、可用、不打扰人的纸。没有账号，没有同步，没有主窗口；待办和笔记直接放在桌面上，随时写、随时折起来。

它是什么

PaperTodo 不是任务管理器，也不是知识库。它更接近桌面上的几张纸：一张写当天待办，一张写临时笔记，需要腾出空间时折成胶囊。

独立纸片窗口

每张纸都是独立无边框窗口，位置、尺寸、置顶和内容会自动保存。

待办纸

一行一个事项，支持勾选、拖动排序、拖拽删除、多行粘贴和撤销重做。

笔记纸

普通文本加轻量 Markdown 显示，适合短笔记、临时内容和桌面备忘。

胶囊模式

纸片可以折叠成小胶囊，并自动停靠到屏幕边缘，需要时点击唤回。

本地数据

数据保存在程序目录的 data.json，并保留备份文件。

源码公开

PaperTodo 免费提供，源码公开用于透明、审计和学习参考。

边界

它刻意不做大而全功能。这个项目的目标是减少桌面临时事项的摩擦，而不是把所有任务都收进一个系统。

适合

  • 当天待办、临时清单、桌面备忘。
  • 想要本地保存、无账号、无同步的小工具用户。
  • 喜欢便签形态，但需要更顺手的待办和轻量 Markdown。

不适合

  • 团队协作、云同步、提醒、日历和统计。
  • 完整 Markdown 编辑器、图片、表格和附件。
  • 分类、标签、搜索、归档和集中任务管理页。
