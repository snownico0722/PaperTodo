PaperTodo 不想管理你的生活。它只是几张安静的桌面纸片 — 想记就写，完成就勾掉，用不着时折叠成胶囊贴到屏幕边上。没有账号，没有云，没有提醒，只有纸。

PaperTodo doesn't want to manage your life. It's just a few quiet sheets of paper on your desktop — jot it down, check it off, fold it into a capsule when you don't need it. No accounts, no cloud, no reminders. Just paper.

用得上时是纸，用不上时是胶囊

Papers when you need them, capsules when you don't

PaperTodo 的胶囊不是"最小化"。它是一整套桌面占用管理：纸片折叠为小胶囊，自动贴到屏幕边缘排队；鼠标靠近时滑出预览，点击回到完整纸片。保持桌面便签的存在感，又不长期霸占空间。

Capsules in PaperTodo aren't "minimize." They're full desktop-space management: papers fold into tiny capsules, auto-dock to screen edges in queues, slide out on hover, expand on click. Always present, never in the way.

  • 普通纸片 → 折叠胶囊 → 贴边胶囊，一套连贯工作流
  • Regular paper → collapsed capsule → edge-docked capsule — one smooth flow
  • 每个屏幕、每个边缘独立排队，拖动就能换边换屏
  • Per-screen, per-edge queues — drag any capsule to a different side or monitor
  • 顶部主胶囊一键收起 / 展开整列贴边队列
  • Master capsule collapses or expands a whole edge queue at once
  • 全屏游戏 / 演示时可选自动避让，不遮挡
  • Optional auto-hide during fullscreen apps — never blocks your content

脚本胶囊：笔记也能一键运行

Script capsules: run notes with one click

在笔记第一行写 !p，纸片就变成一个 PowerShell 脚本。折叠成胶囊后显示闪电图标，左键一键运行，右键打开编辑。把日常脚本、自动化片段变成桌面上的快捷按钮。

Type !p on the first line of a note, and it becomes a PowerShell script. Fold it into a capsule — click to run, right-click to edit. Turn everyday scripts into one-click desktop buttons.

  • 支持 PowerShell 5 和 7，可选隐藏运行窗口
  • Supports PowerShell 5 and 7, with optional hidden execution window
  • 使用 !pf 可复用常驻 PowerShell 进程，秒级响应
  • Use !pf to reuse a persistent PowerShell process for instant response
  • 待办项关联脚本胶囊后，可直接从待办运行
  • Link a script capsule to a todo item to run it directly from the todo

常见问题

FAQ

被问得最多的几个问题。

The questions people ask most.

免费吗？要联网吗？Is it free? Does it need internet?
完全免费，开源。不需要联网，也不需要注册。纸片保存在 data.json，笔记图片保存在同目录的 note-assets.lmdb。
Completely free and open source. No internet or account is required. Papers are stored in data.json and note images in note-assets.lmdb beside it.
支持哪些 Windows 版本？Which Windows versions are supported?
Windows 10 和 11 的 64 位系统。提供两种单文件 exe：自包含版自带 .NET Runtime，无需额外安装；no-runtime 版更小，需要系统已有 .NET 10。
Windows 10 and 11 x64. Two builds are provided: a self-contained version that bundles the .NET runtime (just run it), and a smaller no-runtime version for systems that already have .NET 10.
和 Windows 自带便签有什么不同？How is it different from Windows Sticky Notes?
没有统一列表，每张纸是独立窗口。有专门的待办纸；可折叠为贴边胶囊；不依赖微软账号和云同步；绿色免安装，可放进 U 盘使用。
No central list — each note is its own window. Dedicated todo paper. Papers fold into edge-docked capsules. No Microsoft account or cloud sync. Portable — runs from a USB stick.
数据怎么备份和迁移？How do I back up or migrate my data?
先从托盘退出 PaperTodo，再复制整个文件夹。需要单独备份数据时，请同时保存 data.json、data.backup.json 和 note-assets.lmdb。
Exit PaperTodo from the tray before copying the folder. For a data-only backup, keep data.json, data.backup.json, and note-assets.lmdb together.
首次运行 SmartScreen 拦截怎么办？SmartScreen blocks it on first run?
因为没有购买代码签名证书，首次运行可能弹出 SmartScreen 提示。点击「更多信息 → 仍要运行」即可。每台电脑只需确认一次。
Because the app isn't code-signed, SmartScreen may warn on first run. Click "More info → Run anyway." You only need to do this once per machine.
可以开机自启动吗？Can it start with Windows?
可以。在托盘右键菜单的设置里打开「开机自启动」选项即可。
Yes — enable "Start with Windows" in the tray menu settings.
