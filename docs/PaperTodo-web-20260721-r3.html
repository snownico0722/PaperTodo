02 / 边缘胶囊Edge capsules

用得上时是纸，用不上时是胶囊。

Paper when needed.A capsule when not.

胶囊不是一张伪装的预览卡片，而是纸片真正缩成的轻量窗口。它自动加入屏幕边缘队列，平时只露出一小段；鼠标靠近时滑出，点击恢复完整纸片。

A capsule is not a preview card. It is the paper itself reduced to a lightweight window, queued at the screen edge, sliding out on hover and restoring on click.

  • 01一键让出桌面，需要时点击即回到原位Clear the desktop in one click, then restore in place
  • 02每个屏幕、左右两侧都有独立队列Independent left/right queues on every display
  • 03拖动胶囊即可换边、换屏、重新排序Drag to change side, display, or queue order
  • 04全屏游戏或演示时自动避让Steps aside for fullscreen work

04 / 脚本胶囊Script capsules

把重复动作，
折成一次点击。

Fold repetition
into one click.

笔记首行写入 !p 或 !pf，后续内容就成为 PowerShell 脚本。折叠后显示闪电胶囊：左键运行，右键回到纸片继续编辑。

Start a note with !p or !pf and the rest becomes a PowerShell script. Fold it into a lightning capsule: left-click to run, right-click to edit.

  • 01仍然是普通文本，不制造新的脚本管理器Still plain text, not another script manager
  • 02可选择 PowerShell 7 与常驻进程PowerShell 7 and persistent process options
  • 03可与待办关联，从任务行直接执行Link to a todo and run from its task row

有意保持简单

Intentionally simple

它不想管理你的生活。

It does not want to manage your life.

没有分类、标签、项目树、统计、提醒或日历。PaperTodo 只处理桌面上此刻需要看见的内容。需要团队协作和完整知识库时，请使用更适合的工具。

No categories, tags, project trees, statistics, reminders, or calendars. PaperTodo handles only what deserves a place on your desktop right now.

Q / A

常见问题

Common questions

免费吗？需要联网吗？Is it free? Does it need internet?

完全免费并开源。日常使用不需要联网，也不需要注册账号。

It is completely free and open source. Daily use requires neither internet nor an account.

支持哪些 Windows 版本？Which Windows versions are supported?

支持 Windows 10 与 Windows 11 x64。自包含版本无需另装 .NET。

Windows 10 and Windows 11 x64. The self-contained build needs no separate .NET installation.

数据如何备份和迁移？How do I back up or move my data?

退出 PaperTodo 后复制整个程序文件夹即可；纸片、设置和笔记图片会一起带走。

Exit PaperTodo and copy the whole app folder. Papers, settings, and note images move together.

为什么首次运行会出现 SmartScreen？Why does SmartScreen appear the first time?

项目暂未购买商业代码签名证书。选择“更多信息 → 仍要运行”即可，每台电脑通常只需确认一次。

The project does not yet use a commercial code-signing certificate. Choose “More info → Run anyway.”
